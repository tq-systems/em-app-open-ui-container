#############
Weboberfläche
#############

Die Weboberfläche ermöglicht dem Nutzer die Konfiguration des Gerätes mit einem Browser durch die Webseite.
Dabei ist die Verwendung von javascript nötig und darf zur korrekten Funktionsweise nicht geblockt werden.
Die Weboberfläche ist vor ungewolltem Zugriff durch ein Passwort geschützt.
Das Erstpasswort ist auf dem Etikett des Gerätes aufgedruckt und sollte nach dem ersten Einloggen geändert werden.

=====
Login
=====

Auf der Loginseite wird das Produktlogo sowie eine Willkommensnachricht angezeigt.
Darunter befindet sich ein Eingabefeld für das Passwort des Energy Managers und ein **Einloggen** Button.
Um Zugang zur Konfiguration des Gerätes zu erlangen, bitte das Passwort eingeben und auf **Einloggen** klicken.

=========
Dashboard
=========

Das Dashboard zeigt eine Übersicht über das System in Form von Dashboard-Widgets, sowie möglicherweise eine Übersicht
aller installierten Apps.
Zudem enthält es diverse Steuerelemente.

Kopfzeile
------

Auf der linken Seite befindet sich eine Navigationsleiste. Mit ihrer Hilfe kann zwischen der ausgewählten Anwendung gewechselt werden.
Es kann sein, dass die Navigationsleiste zunächst durch einen Klick das Symbol mit den 3 horizontalen Strichen geöffnet werden muss.


Systemnachrichten
~~~~~~~~~~~~~~~~~

Zeigt die Anzahl ungelesener Systembenachrichtigungen. Ein Klick auf das Glockensymbol öffnet die Health-Check App.
Diese dient zur Anzeige von Meldungen der Kategorien Information, Fehler und Warnung.

Spracheinstellung
~~~~~~~~~~~~~~~~~

Ändern der Sprache durch Auswahl der gewünschten Option.

Profil
~~~~~~

Über die Schaltfläche Profil gelangt man zu den Optionen **Ausloggen**, **Passwort ändern** und **Zugriffsschlüssel**.

- **Ausloggen** Meldet den Benutzer ab und leitet auf die Login Seite weiter. Die Weboberfläche bleibt bis zum erneuten Einloggen gesperrt.
- **Passwort ändern** Öffnet die Schaltfläche zur Änderung des Passworts (dringend empfohlen nach Erstanmeldung).
- **Zugriffsschlüssel** Öffnet die Schaltfläche zur Erstellung und Autorisierung von Zugriffschlüsseln.

Fußzeile
------

In der Fußzeile lassen sich die mittels **Lizenzen** die Lizenzen und rechtlichen Informationen anzeigen.
Außerdem können dort System-Lizenzen und App-Lizenzen durch drücken von **Herunterladen** auf dem verwendeten Gerät gespeichert werden.

========
Passwort
========

Nach dem Klick auf **Passwort ändern** öffnet sich ein Fenster zur Änderung des Passwortes. Dort muss zunächst das aktuelle Passwort eingegeben werden.
Anschließend kann ein neues Passwort vergeben werden.

Dabei ist darauf zu achten, dass das neue Passwort folgenden Richtlinien entspricht:

- Passwort muss mindestens 12 Zeichen lang sein
- Passwort muss mindestens einen Großbuchstaben enthalten
- Passwort muss mindestens eine Zahl enthalten

Abschließend das Passwort unter **Bestätigung** erneut eingeben und mit dem Klick auf **Senden** setzen.
Sollte das aktuelle Passwort nicht korrekt sein, so wird "Ungültiges Passwort" angezeigt und die Eingabe lässt sich bearbeiten.
Nach dem erfolgreichen Senden des Passwortes ist nur das neue gültig und muss zur Anmeldung verwendet werden.
